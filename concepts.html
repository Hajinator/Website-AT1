Authentication is the process of verifying the identity of a user or a system, usually through credentials like usernames and passwords. It ensures that only authorized individuals can access resources or perform specific actions.
Web Security refers to the measures used to protect websites from threats and vulnerabilities. It includes implementing cyber security protocols, technologies and practises to safeguard data and prevent unauthorized access.

HTTP is a protocol used for transmitting hypertext (web) documents over the internet. It defines how messages are formatted and transmitted, and how web servers and browsers respond to various commands.

Session management is the process of handling user sessions on a website. It involves creating, maintaining and terminating sessions to track user interactions and preferences. This includes storing session data and ensuring that sessions are properly managed.

Stateless programming is when each request or interaction is treated independently, without relying on prior interactions or maintaining a state between them.
