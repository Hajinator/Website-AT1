The three major programming control structures involve:
Sequence: The sequence control structure involves executing instructions one after the other in a linear flow, from the beginning of the program to the end.
Selection:
The selection control structure allows a program to choose between different paths based on the conditions in the program. This is implemented using conditional statements such as ‘if’, ‘else if’, ‘else’ or switch case statements. Iteration:
The iteration control structure allows the repetition of code until a certain condition is met. This is implemented using iteration statements such as ‘for’, ‘while’, and ‘do-while’.

HTML (Hypertext Markup Language) is the standard language used to create and structure content on the web. It used tags and attributes to define elements like headings, paragraphs, links, images and other content. Another way to think of HTML is almost like the “skeleton” of the webpage.

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS controls the layout, color, font, spacing and overall visual appearance of a web page.

Syntax refers to the rules that define the structure of a programming language. These rules specify how naming conventions, keywords, symbols and characters must be used to create a program that runs effectively and to those languages standards.
