Programming controls refer to constructs that manage the flow of execution in a program.
This includes statements like loops (‘for, ‘while’), conditionals (‘if’, switch’) and branching commands (‘continue’, ‘break’).
These controls help direct different parts of the code which are executed based on certain conditions.
Design structures are patterns used to organize code and include concepts such as:
Sequential: Executing code in a linear order.
Selection: Making decisions in code, such as if statements.
Iteration: Repeating code blocks, such as loops.
Modularization: Breaking code into smaller, reusable functions or modules.

There are a variety of ways to test websites such as:
Functional Testing: Verifying that all functions of the website work properly.
Usability: Assessing how user-friendly the website is.
Compatibility: Checking how the website performs across different browsers and devices.
Performance: Measuring the website's load time and responsiveness under various conditions.
Security: Identifying cyber security threats.

There are a variety of ways to debug websites such as:
Console Logging: Using console log statements to track code execution and variable values.
Developer Tools: Using built-in webpage tools to inspect elements and analyse performance.
Error Messages: Reading and interpreting error messages to understand and resolve issues.
Different Environments: Checking the website on various devices and browsers to find and fix specific issues.
Version Control: Using version control systems to track changes and identify when and where bugs were introduced.

There are a variety of coding techniques such as:
Responsive Design: Using CSS media queries and flexible layouts to ensure the website adapts to different screen sizes.
Modular Coding: Writing code in small, reusable modules or components to enhance maintainability.
Semantic HTML: Using HTML tags that convey the meaning of the content such as article, section, header tags.
Progressive Enhancement: Building a basic level of functionality that works for all users and then enhancing it for those with more advanced browsers or devices.
Optimization: Minimizing and compressing code, images, and assets to improve website performance and loading times.
